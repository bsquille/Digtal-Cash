Readme for Project 3 - Digital Cash

Authors - Lev Berzon-Kozlov and Brad Squillets

Version - 1.0

Contact information - lberzonk@emich.edu or bsquille@emich.edu


Dependencies - Must have Powershell 5.0 or higher installed. Can be installed at https://github.com/PowerShell/PowerShell. Must have Perl installed on machine. For Linux or Mac machines,
make sure that Perl is up to date. For Windows machines, Perl can be found at https://www.perl.org/get.html.



Program Description  - This program will allow a user to implement a demonstartion of the Digital Cash concept as presented by protocol number 4, described in [SCHN96], p. 142. Program could be run by right clicking the .ps1
file from explorer.
